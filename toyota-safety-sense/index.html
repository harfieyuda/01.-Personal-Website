Toyota Safety Sense
Project Type
Interactive Installation
Project Year
2022

Toyota Safety Sense is an interactive installation that provide information of safety features on Toyota Vehicles during driving.

The main goal of this interactive application is to introduce the safety features on Toyota Vehicles to people who want to buy a vehicle. To help understanding the information easily, this interaction installation also provides videos that showing how the safety features work in entertainment way.

Since the main goal of this application is to provide information, a clean design is important to achieve the information hierarcy and consistency of look. Furthermore, using gradient colors as main colors are to give a futuristic impression of Toyota Vehicle since the main key visual of this application is Hi-Tech Design.
