Samsung x BMW Promotion 2024
Project Type
3D Motion Graphic
Project Year
2024

Samsung x BMW is a collaboration between 2 brands to promote the launching of new Galaxy Z Fold 6 and Galaxy Z Flip 6.

During this project, i had 3 tasks to do. First task was i had to create all of BMW i5 Scene for the social media promotion. Since the project timeline was very short, i had to use unreal engine 5 to create the environment and animate the scene. I also use after effect to create circuit path effect.

Second task, i was using blender 3d to create a phone animation for the anamorphic led animation while my co-worker create the bmw scene. I also composite all of the 3d scene and making the anamorphic effects.

The last task was i had to create a booting animation for Galaxy Z Fold 6. Since this was a collaboration between samsung and mandiri prioritas, the luxury looks and flying cloth was the main approach for the animation.
